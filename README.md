# todolist

파이썬으로 만든 기본 CLI To Do List 프로그램입니다. 팀 프로젝트를 처음 시작하는 단계에서 읽고 수정하기 쉽도록 핵심 기능과 화면 입출력을 한 파일 안에서 작게 구성했습니다.

## 기능

- 할 일 목록 보기
- 할 일 추가
- 할 일 수정
- 할 일 삭제
- 완료/미완료 변경
- `todos.json` 파일 저장

## 실행 방법

```bash
python3 todo.py
```

## 파일 구조

```text
todo.py      # 프로그램 코드
todos.json   # 실행 후 자동 생성되는 저장 파일
README.md    # 프로젝트 설명
```
